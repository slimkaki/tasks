# Tasks

Projeto de um CRUD para tasks (ou tarefas), sendo uma API REST. Projetado em Django e interação com um banco de dados Postgresql.

## Endpoints
<li>
<ul>/get_all</ul></br>
<ul>/get_single/{id_task}</ul></br>
<ul>/post</ul></br>
<ul>/delete/{id_task}</ul></br>
<ul>/update/{id_task}</ul></br>
<ul>/delete_all</ul></br>
</li>
